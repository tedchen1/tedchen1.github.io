/Like 操作/
select from Test where name like ‘%t%’
select from Test where name not like ‘%t%’/not like/

/IN 操作/
select from Test where name IN(‘make’,’lina’)
select from Test where name =’make’ or name=’lina’

/between…and操作/
select * from Test where name between ‘make’ and ‘ted’

/使用表别名/
select a.name,a.sex,a.id from Test as a where a.name=’ted’

select name from Test left join Test1 on Test.id=Test1.sid

关于共享单车机制完善的思考

共享单车的普及同时也是一场对人性的考验
当你把共享单车藏进楼道的那一刻，当你把私锁加上去的那一刻，人们内心中自私贪婪的阴暗面暴露无遗，社会上的监管力度和单车公司
的触发力度几乎形同虚设，在我眼里OFO简直就是慈善公司，各种BUG，无定位，一码一车，简直实在给街上的“中年老大妈”们送温暖，
对于这种现象，我构思了一下关于共享单车对于违规使用的处理办法，下面以ofo和小蓝单车为例
1.所有单车必备定位系统（不仅是为了解决私有化问题，更是方便管理）
2.处罚措施
（1）在扫码处旁边增加举报功能，后台判断（if）当前车辆被五个不用用户举报切举报间隙无新用户使用，则不问理
由直接封禁第一个举报前最后一位使用者帐号扣押保证金。
（2）（if）同一账号在每天固定时段使用车辆超过三天，（or）或三天内只有单个用户使用，期间无新用户使用（针对OFO）
则视作单车私有化，直接封禁帐号扣押保证金。
（3）（if）单车定位凡是显示与居民楼重合（市场上的地图可以实现该功能）视作单车违停，直接封禁帐号扣押保证金。
（4）一经调查确认违规上私锁者，讲交由公安机关进行行政处罚。（已有相关规定）
3.关于现有的信誉积分主要作为奖励措施，举报一经核实处罚，举报人得到相应信誉积分，信誉积分可用作兑换代金券。
4.关于具体的处罚措施，第一次封停帐号一个月保证金不退回，第二次永久封号，并上征信黑名单。123
